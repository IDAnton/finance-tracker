# Finance-tracker


RESTful API для учета финансов

## 🛠 Технологический стек
* **ЯП**: Java 21
* **Фреймворк**: Spring Boot 4.1.0
* **Аутентификация**: Spring Security, JWT, BCrypt
* **База данных**: PostgreSQL, H2 (In-Memory для тестирования)
* **Миграции**: Liquibase
* **Тестирование**: JUnit 5, Mockito: юнит тесты, интеграционные тесты


## ⚙️ Локальный запуск
1. Клонировать репозиторий: `git clone https://github.com/IDAnton/finance-tracker`
2. Запустить docker-compose: `docker-compose up -d`
